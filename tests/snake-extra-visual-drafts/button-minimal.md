# Button - Minimal Template

## Component Metadata

### Name

Button

### Description

An interactive element that triggers an action or event.

---

## Visual Structure

Inline layout with centered content.

### Content Areas or Slots

- **Label (required):** The text describing the action.

---

## Variants

- **Primary:** Filled solid background with primary color
- **Secondary:** Muted fill color

---

## States

### Default

Base appearance with standard styling.

### Hover

Visual feedback when pointer hovers over the button.
