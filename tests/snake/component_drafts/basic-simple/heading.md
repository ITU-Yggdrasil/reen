# Heading

## Description
Heading is the primary typographic component for titles and section labels that establish hierarchy and structure. Use it to signal content organization and page flow, and avoid using it for decorative emphasis where no structural meaning exists.

## Purpose
The purpose of Heading is to establish hierarchy, introduce sections, and guide content scanning.

## Visual Structure
Heading is a bold, prominent text block with predictable spacing above and below. It should anchor the surrounding content and create a clear visual break between sections without overwhelming the layout.

## Subcomponents
None.

## States & Variants
- States: default and inverted when displayed on dark surfaces.
- Variants: display, page, section, subsection, card title.
- Alignment options: left aligned by default with centered usage only when layout intent is explicit.

## Properties
- `content`: visible title text.
- `level`: hierarchy depth.
- `variant`: stylistic scale.
- `alignment`: text placement.
- `truncate`: whether long titles may be shortened in constrained contexts.
- `subtext`: optional short supporting line when paired intentionally.

## Brand Reference
No formal brand specification is required for this draft. Heading should use strong, clean typography and clear spacing to establish hierarchy. Emphasis should come from scale and readability, while decorative or overly expressive treatment should be avoided.
