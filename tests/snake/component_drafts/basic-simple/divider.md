# Divider

## Description
Divider is a structural separator used to group, split, or pace related content. Use it to clarify transitions between sections or dense information groups, and avoid using it where spacing alone already provides adequate separation.

## Purpose
The purpose of Divider is to separate related content groups and improve structural clarity.

## Visual Structure
Divider is a thin horizontal or vertical rule with optional inset alignment. It should recede into the layout while still providing enough visual structure to organize content.

## Subcomponents
None.

## States & Variants
- States: default and subdued.
- Variants: horizontal, vertical, full-width, inset.

## Properties
- `orientation`: horizontal or vertical.
- `inset`: controls side spacing.
- `tone`: subtle or standard emphasis.
- `length`: full container or custom constrained length.

## Brand Reference
No formal brand specification is required for this draft. Divider should stay subtle and structural, helping organize content without drawing attention to itself. Separation should rely on calm visual rhythm, spacing, and sufficient contrast rather than decorative styling.
